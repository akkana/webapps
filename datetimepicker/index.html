Astronomy Date/Time Picker Widget

I use this mostly for my JavaScript astronomy apps.

Use it in a page: 

<div id="astrotimewidget"></div>
[ ... ]
<script type="text/javascript"
        src="/javascript/datetimepicker/astrotimewidget.js"></script>

Useful functions you might want to put on buttons:
reset2now() Reset to the current time
addHours(hrs) Add time (may be fractional or negative)
getPickerDate() Get the current time as a JS Date object
setPickerDate() Set the picker date to a given Date object

Set a function to be called when the date changes: 

dateChangeCallback = function(d) { ... };

Hide the Universal Time line: 

#astrotimewidget #show-ut { display: none; }

Disable animations with the css rule: 

#astrotimewidget #show-animation { display: none; }

Disable the orientation chooser with the css rule: 

#astrotimewidget #show-orientation { display: none; }
